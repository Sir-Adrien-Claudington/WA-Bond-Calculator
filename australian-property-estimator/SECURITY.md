# Security Policy

## Reporting a Vulnerability

Email **[REDACTED]** with subject line `[newgearing Security]`.

- Acknowledgement target: 72 hours
- Fix or mitigation target: 14 days

Do not open public GitHub issues for security vulnerabilities.

## Scope

This project is a purely client-side calculator. No user data, credentials, or PII are collected or stored. There are no secrets, no database, and no authentication.

## Secrets

This project currently has no secrets. If credentials are ever added (e.g., a backend API key), rotate immediately upon any suspected exposure and document in this file.
